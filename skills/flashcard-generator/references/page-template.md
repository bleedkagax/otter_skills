# Flashcard HTML Page Template

When generating HTML output, produce a **complete self-contained HTML file** with the flashcard data embedded as a JS array. The page renders flip cards with substage grouping, quiz mode, and keyboard navigation.

## Output Rules

- Start with `<!DOCTYPE html>` — no preamble text
- Embed all card data as `const cards = [...]` and `const quizItems = [...]` inside `<script>`
- Use Tailwind CSS via CDN for styling
- Use MathJax via CDN if cards contain math (LaTeX `$...$`)
- Total HTML should be **300-600 lines** depending on card count

## HTML Skeleton

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[TOPIC] — Flashcards</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <!-- Include MathJax ONLY if cards contain math -->
    <script>
        window.MathJax = {
            tex: { inlineMath: [['$', '$'], ['\\(', '\\)']] },
            svg: { fontCache: 'global' }
        };
    </script>
    <script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"></script>
    <style>
        /* Flip card animation */
        .flip-card { perspective: 1000px; cursor: pointer; }
        .flip-inner {
            position: relative; width: 100%; height: 100%;
            transition: transform 0.6s cubic-bezier(0.4, 0, 0.2, 1);
            transform-style: preserve-3d;
        }
        .flip-card.flipped .flip-inner { transform: rotateY(180deg); }
        .flip-front, .flip-back {
            position: absolute; inset: 0;
            backface-visibility: hidden;
            display: flex; align-items: center; justify-content: center;
            padding: 2rem; border-radius: 1rem;
        }
        .flip-back { transform: rotateY(180deg); }

        /* Substage color bands */
        .ss-1 { border-left: 4px solid #3b82f6; } /* blue - definition */
        .ss-2 { border-left: 4px solid #8b5cf6; } /* purple - comparison */
        .ss-3 { border-left: 4px solid #f59e0b; } /* amber - mechanism */
        .ss-4 { border-left: 4px solid #10b981; } /* green - relationship */
        .ss-5 { border-left: 4px solid #ef4444; } /* red - application */

        /* Quiz feedback */
        .quiz-correct { background: #dcfce7 !important; border-color: #22c55e !important; }
        .quiz-wrong { background: #fee2e2 !important; border-color: #ef4444 !important; }

        /* Progress bar */
        .progress-fill { transition: width 0.3s ease; }
    </style>
</head>
<body class="min-h-screen bg-gradient-to-br from-slate-50 to-slate-100">
    <!-- Header -->
    <header class="sticky top-0 z-10 bg-white/80 backdrop-blur border-b border-slate-200 px-6 py-4">
        <div class="max-w-4xl mx-auto flex items-center justify-between">
            <div>
                <h1 class="text-2xl font-bold text-slate-800">[TOPIC]</h1>
                <p class="text-sm text-slate-500" id="progress-text">Card 1 / N</p>
            </div>
            <div class="flex gap-2">
                <button onclick="setMode('cards')" id="btn-cards"
                    class="px-4 py-2 rounded-lg text-sm font-medium bg-slate-800 text-white">
                    Cards
                </button>
                <button onclick="setMode('quiz')" id="btn-quiz"
                    class="px-4 py-2 rounded-lg text-sm font-medium bg-slate-100 text-slate-600 hover:bg-slate-200">
                    Quiz
                </button>
                <button onclick="setMode('grid')" id="btn-grid"
                    class="px-4 py-2 rounded-lg text-sm font-medium bg-slate-100 text-slate-600 hover:bg-slate-200">
                    Grid
                </button>
            </div>
        </div>
        <!-- Progress bar -->
        <div class="max-w-4xl mx-auto mt-2 h-1 bg-slate-200 rounded-full overflow-hidden">
            <div class="progress-fill h-full bg-slate-800 rounded-full" id="progress-bar" style="width: 0%"></div>
        </div>
    </header>

    <!-- Substage filter tabs -->
    <nav class="max-w-4xl mx-auto px-6 pt-4 flex gap-2 flex-wrap" id="filter-tabs">
        <button onclick="filterSubstage(0)" class="px-3 py-1 rounded-full text-xs font-medium bg-slate-800 text-white">All</button>
        <!-- Tabs generated by JS based on available substages -->
    </nav>

    <!-- Card view (single card with flip) -->
    <main class="max-w-4xl mx-auto px-6 py-8" id="card-view">
        <div class="flip-card mx-auto" style="width: 100%; max-width: 640px; height: 360px;" onclick="flipCard()">
            <div class="flip-inner" id="flip-inner">
                <div class="flip-front bg-white shadow-lg border border-slate-200 text-center">
                    <div class="text-lg font-medium text-slate-800" id="card-front">Loading...</div>
                </div>
                <div class="flip-back bg-slate-800 shadow-lg text-center">
                    <div class="text-lg text-white leading-relaxed" id="card-back">Loading...</div>
                </div>
            </div>
        </div>
        <!-- Substage badge -->
        <div class="text-center mt-4">
            <span class="inline-block px-3 py-1 rounded-full text-xs font-medium" id="substage-badge"></span>
        </div>
        <!-- Navigation -->
        <div class="flex justify-center gap-4 mt-6">
            <button onclick="prevCard()" class="px-6 py-3 rounded-xl bg-white border border-slate-200 text-slate-600 hover:bg-slate-50 font-medium">
                ← Prev
            </button>
            <button onclick="nextCard()" class="px-6 py-3 rounded-xl bg-slate-800 text-white hover:bg-slate-700 font-medium">
                Next →
            </button>
        </div>
        <p class="text-center text-xs text-slate-400 mt-3">Click card to flip · Arrow keys to navigate · Space to flip</p>
    </main>

    <!-- Quiz view -->
    <main class="max-w-4xl mx-auto px-6 py-8 hidden" id="quiz-view">
        <div class="bg-white rounded-2xl shadow-lg border border-slate-200 p-8" id="quiz-container">
            <div class="text-sm text-slate-400 mb-2" id="quiz-progress">Question 1 / N</div>
            <h2 class="text-xl font-bold text-slate-800 mb-6" id="quiz-question">Loading...</h2>
            <div class="space-y-3" id="quiz-options"></div>
            <div class="mt-6 hidden" id="quiz-explanation">
                <div class="p-4 rounded-xl bg-blue-50 border border-blue-200 text-blue-800 text-sm" id="quiz-explain-text"></div>
                <button onclick="nextQuiz()" class="mt-4 px-6 py-3 rounded-xl bg-slate-800 text-white hover:bg-slate-700 font-medium">
                    Next Question →
                </button>
            </div>
        </div>
    </main>

    <!-- Grid view -->
    <main class="max-w-4xl mx-auto px-6 py-8 hidden" id="grid-view">
        <div class="grid grid-cols-1 sm:grid-cols-2 gap-4" id="grid-container"></div>
    </main>

    <script>
    // ========== DATA (AI fills this) ==========
    const cards = [
        // { front: "...", back: "...", substage: 1, label: "Definition" }
    ];
    const quizItems = [
        // { question: "...", options: ["A","B","C","D"], answer_index: 0, explanation: "..." }
    ];

    // ========== SUBSTAGE CONFIG ==========
    const ssColors = {
        1: { bg: 'bg-blue-100', text: 'text-blue-700', border: 'border-blue-400', label: 'Definition' },
        2: { bg: 'bg-purple-100', text: 'text-purple-700', border: 'border-purple-400', label: 'Comparison' },
        3: { bg: 'bg-amber-100', text: 'text-amber-700', border: 'border-amber-400', label: 'Mechanism' },
        4: { bg: 'bg-emerald-100', text: 'text-emerald-700', border: 'border-emerald-400', label: 'Relationship' },
        5: { bg: 'bg-red-100', text: 'text-red-700', border: 'border-red-400', label: 'Application' }
    };

    // ========== STATE ==========
    let currentIndex = 0;
    let currentMode = 'cards';
    let filteredCards = [...cards];
    let activeFilter = 0;
    let quizIndex = 0;
    let quizAnswered = false;

    // ========== CARD MODE ==========
    function showCard(idx) {
        currentIndex = idx;
        const card = filteredCards[idx];
        if (!card) return;
        document.getElementById('card-front').innerHTML = card.front;
        document.getElementById('card-back').innerHTML = card.back;
        document.getElementById('flip-inner').parentElement.classList.remove('flipped');

        const ss = ssColors[card.substage] || ssColors[1];
        const badge = document.getElementById('substage-badge');
        badge.className = `inline-block px-3 py-1 rounded-full text-xs font-medium ${ss.bg} ${ss.text}`;
        badge.textContent = `${ss.label} (${card.substage}/5)`;

        document.getElementById('progress-text').textContent = `Card ${idx + 1} / ${filteredCards.length}`;
        document.getElementById('progress-bar').style.width = `${((idx + 1) / filteredCards.length) * 100}%`;

        if (window.MathJax) MathJax.typesetPromise();
    }

    function flipCard() {
        document.getElementById('flip-inner').parentElement.classList.toggle('flipped');
    }
    function prevCard() { if (currentIndex > 0) showCard(currentIndex - 1); }
    function nextCard() { if (currentIndex < filteredCards.length - 1) showCard(currentIndex + 1); }

    // ========== FILTER ==========
    function filterSubstage(ss) {
        activeFilter = ss;
        filteredCards = ss === 0 ? [...cards] : cards.filter(c => c.substage === ss);
        currentIndex = 0;
        if (filteredCards.length) showCard(0);
        buildGrid();

        document.querySelectorAll('#filter-tabs button').forEach((btn, i) => {
            if ((ss === 0 && i === 0) || (ss > 0 && i === ss)) {
                btn.className = btn.className.replace(/bg-slate-100 text-slate-600/, 'bg-slate-800 text-white');
            } else {
                btn.className = btn.className.replace(/bg-slate-800 text-white/, 'bg-slate-100 text-slate-600');
            }
        });
    }

    function buildFilterTabs() {
        const tabs = document.getElementById('filter-tabs');
        const substages = [...new Set(cards.map(c => c.substage))].sort();
        substages.forEach(ss => {
            const s = ssColors[ss] || ssColors[1];
            const btn = document.createElement('button');
            btn.className = 'px-3 py-1 rounded-full text-xs font-medium bg-slate-100 text-slate-600 hover:bg-slate-200';
            btn.textContent = s.label;
            btn.onclick = () => filterSubstage(ss);
            tabs.appendChild(btn);
        });
    }

    // ========== QUIZ MODE ==========
    function showQuiz(idx) {
        quizIndex = idx;
        quizAnswered = false;
        const q = quizItems[idx];
        if (!q) return;
        document.getElementById('quiz-progress').textContent = `Question ${idx + 1} / ${quizItems.length}`;
        document.getElementById('quiz-question').innerHTML = q.question;
        document.getElementById('quiz-explanation').classList.add('hidden');

        const optCont = document.getElementById('quiz-options');
        optCont.innerHTML = '';
        q.options.forEach((opt, i) => {
            const btn = document.createElement('button');
            btn.className = 'w-full text-left px-4 py-3 rounded-xl border border-slate-200 bg-white hover:bg-slate-50 text-slate-700 font-medium transition-all';
            btn.innerHTML = `<span class="text-slate-400 mr-2">${String.fromCharCode(65 + i)}.</span> ${opt}`;
            btn.onclick = () => answerQuiz(i, q.answer_index, q.explanation);
            optCont.appendChild(btn);
        });

        if (window.MathJax) MathJax.typesetPromise();
    }

    function answerQuiz(chosen, correct, explanation) {
        if (quizAnswered) return;
        quizAnswered = true;
        const btns = document.querySelectorAll('#quiz-options button');
        btns[correct].classList.add('quiz-correct');
        if (chosen !== correct) btns[chosen].classList.add('quiz-wrong');

        const expDiv = document.getElementById('quiz-explanation');
        document.getElementById('quiz-explain-text').innerHTML = explanation;
        expDiv.classList.remove('hidden');
        if (window.MathJax) MathJax.typesetPromise();
    }

    function nextQuiz() {
        if (quizIndex < quizItems.length - 1) showQuiz(quizIndex + 1);
    }

    // ========== GRID MODE ==========
    function buildGrid() {
        const cont = document.getElementById('grid-container');
        cont.innerHTML = '';
        filteredCards.forEach((card, i) => {
            const ss = ssColors[card.substage] || ssColors[1];
            const div = document.createElement('div');
            div.className = `p-5 rounded-xl border bg-white shadow-sm cursor-pointer hover:shadow-md transition-shadow ss-${card.substage}`;
            div.innerHTML = `
                <span class="inline-block px-2 py-0.5 rounded text-xs font-medium mb-2 ${ss.bg} ${ss.text}">${ss.label}</span>
                <div class="font-medium text-slate-800 mb-2">${card.front}</div>
                <div class="text-sm text-slate-500 hidden card-answer">${card.back}</div>
            `;
            div.onclick = () => {
                div.querySelector('.card-answer').classList.toggle('hidden');
                if (window.MathJax) MathJax.typesetPromise();
            };
            cont.appendChild(div);
        });
    }

    // ========== MODE SWITCH ==========
    function setMode(mode) {
        currentMode = mode;
        document.getElementById('card-view').classList.toggle('hidden', mode !== 'cards');
        document.getElementById('quiz-view').classList.toggle('hidden', mode !== 'quiz');
        document.getElementById('grid-view').classList.toggle('hidden', mode !== 'grid');

        ['cards', 'quiz', 'grid'].forEach(m => {
            const btn = document.getElementById(`btn-${m}`);
            if (m === mode) {
                btn.className = btn.className.replace(/bg-slate-100 text-slate-600/, 'bg-slate-800 text-white');
            } else {
                btn.className = btn.className.replace(/bg-slate-800 text-white/, 'bg-slate-100 text-slate-600');
            }
        });

        if (mode === 'quiz' && quizItems.length) showQuiz(0);
        if (mode === 'grid') buildGrid();
    }

    // ========== KEYBOARD ==========
    document.addEventListener('keydown', e => {
        if (currentMode === 'cards') {
            if (e.key === 'ArrowRight') nextCard();
            else if (e.key === 'ArrowLeft') prevCard();
            else if (e.key === ' ') { e.preventDefault(); flipCard(); }
        }
    });

    // ========== INIT ==========
    buildFilterTabs();
    if (cards.length) showCard(0);
    </script>
</body>
</html>
```

## Data Format

The AI fills `const cards` and `const quizItems`:

```javascript
const cards = [
    { front: "What is Ohm's Law?", back: "V = IR; voltage equals current times resistance.", substage: 1, label: "Definition" },
    { front: "Compare series and parallel circuits", back: "Series: single path, current same. Parallel: multiple paths, voltage same.", substage: 2, label: "Comparison" },
    // ...
];

const quizItems = [
    { question: "What does R represent in V = IR?", options: ["Voltage", "Resistance", "Current", "Power"], answer_index: 1, explanation: "R is resistance, measured in ohms." },
    // ...
];
```

## Substage Labels

| Substage | STEM Label | Humanities Label | Color |
|----------|-----------|-----------------|-------|
| 1 | Definition | Nouns | Blue |
| 2 | Comparison | Descriptors | Purple |
| 3 | Mechanism | Actions | Amber |
| 4 | Relationship | Functions | Green |
| 5 | Application | Expressions | Red |

For Humanities, update the `ssColors` labels to match:
```javascript
const ssColors = {
    1: { ..., label: 'Nouns' },
    2: { ..., label: 'Descriptors' },
    // etc.
};
```
